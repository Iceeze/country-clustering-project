# Кластеризация стран по социально-экономическим показателям

---

## Тема проекта

Анализ и группировка стран на основе их социально-экономических характеристик с использованием методов машинного обучения (кластеризация).

## Цель проекта

Выявить скрытые закономерности и объединить страны в кластеры по уровню развития, благосостояния, здоровья населения и другим показателям. Результаты помогут понять, какие страны схожи между собой, и выделить уникальные регионы для дальнейшего социально-экономического анализа.

## Источник данных

*Датасет: World Happiness Report или Country Statistics.*
